# -AI-Based-Email-Spam-Phishing-Detector
An intelligent cybersecurity project that detects spam and phishing emails using Machine Learning, Python, Flask, and NLP techniques. The system analyzes email content and classifies it as Spam/Phishing or Legitimate in real time.

🚀 Features


Detects spam and phishing emails instantly

Uses TF-IDF Vectorization for text processing

Machine Learning model using Naive Bayes

Simple and interactive Flask Web Interface

Real-time email text prediction

Lightweight and beginner-friendly cybersecurity project

🛠️ Technologies Used

Python

Flask

Scikit-learn

TF-IDF Vectorizer

Naive Bayes Algorithm

HTML/CSS

📂 Project Workflow


Collect email dataset

Convert text into numerical vectors using TF-IDF

Train ML model using Naive Bayes

Save trained model and vectorizer

Predict whether an email is spam/phishing or safe

🎯 Use Case



This project can help users identify suspicious emails that may attempt to steal sensitive 
information such as passwords, bank details, or personal data.



▶️ Run the Project

pip install -r requirements.txt

python train_model.py

python app.py
